Projects in the Lab

 

Code Name: LoreCraft

LoreCraft is derived from "lore," which refers to storytelling, myth, and traditional tales, and "craft," which signifies the creation and skillful development of something new. This name represents the app’s mission: to skillfully craft personalized, imaginative stories (or "lore") for each child, making storytime unique, interactive, and engaging.

Project Details: LoreCraft is an AI-powered mobile app that generates personalized bedtime stories for children. By inputting their child’s name, age, and favorite themes, parents can instantly create unique, engaging tales crafted to their child’s interests. LoreCraft offers daily story suggestions, customizable story lengths, and an interactive “choose-your-own-adventure” option to make each story unforgettable. With LoreCraft, storytime becomes a magical, educational experience, supporting language and cognitive growth through imaginative, personalized storytelling.

  • Backend and API Integration:
    • Language: Python (Flask/Django) or Node.js for API handling.
    • Database: Firebase or AWS DynamoDB for storing user profiles, preferences, and story histories.
    • Cloud Functions: AWS Lambda or Firebase Functions for serverless compute tasks.
  •  
  • Frontend Development:
    • Framework: Flutter or React Native for cross-platform mobile development (iOS and Android).
    • UI/UX Libraries: Material-UI, React Navigation (React Native), or Rive for interactive elements.
    • Media: Integration with TTS services (Google Text-to-Speech or Amazon Polly) for story narration.
  •  
  • AI and Natural Language Processing:
    • NLP Model: OpenAI’s GPT model for generating unique, context-based story content.
    • Text-to-Speech: Google Cloud TTS or Amazon Polly for multi-voice and multilingual support.
    • Sentiment Analysis: (Optional) Implement sentiment libraries to adjust storytelling tone (e.g., VADER or TextBlob for Python).

Code Name: SpiceSpark

SpiceSpark blends "spice" and "spark," symbolizing the app's mission to ignite creativity in the kitchen by transforming ordinary ingredients into exciting meals. SpiceSpark empowers users to rediscover their pantry’s potential, sparking culinary inspiration and helping reduce food waste.

Project Details: SpiceSpark is an AI-powered mobile app that generates custom recipes based on ingredients users already have at home. By simply entering their pantry items, users receive unique recipes tailored to their preferences, dietary needs, and meal types (breakfast, lunch, dinner). With daily recipe inspiration and hands-free cooking options, SpiceSpark brings convenience and culinary exploration to every kitchen.

  • Backend and API Integration:
    • Language: Python (Flask/Django) or Node.js for API management.
    • Database: Firebase or AWS DynamoDB to store user profiles, favorite recipes, and ingredient lists.
    • Cloud Functions: AWS Lambda or Firebase Functions to process ingredients and generate recipes on demand.
  •  
  • Frontend Development:
    • Framework: Flutter or React Native for a smooth, cross-platform mobile experience.
    • UI/UX Libraries: Material-UI and React Navigation (React Native) for intuitive browsing and recipe selection.
    • Media Integration: Google Text-to-Speech or Amazon Polly to read recipes aloud, enabling hands-free cooking.
  •  
  • AI and Natural Language Processing:
    • NLP Model: OpenAI’s GPT model to create recipes based on ingredient input and dietary specifications.
    • Ingredient Recognition: Implement NLP to adjust recipes based on available ingredient quantities and suggest substitutes.
    • Personalization: Machine learning to recommend recipes based on user preferences and frequently used ingredients.
